Raspberry Pi Setup

One-Step Installation for FFGSA_CSM Player

What This Does

  • Installs all required software automatically
  • Downloads and configures the video player
  • Registers your device with the server
  • Sets up automatic startup on boot
  • Optimizes display for digital signage
  • Starts playing content immediately

Requirements

Raspberry Pi
3B+ or newer
Raspberry Pi OS
Bullseye or newer
Network
WiFi or Ethernet
Display
HDMI connected

Installation Steps

1
Open Terminal on Your Raspberry Pi

Click the terminal icon in the menu bar or press Ctrl+Alt+T

2
Run the Installation Command

Copy and paste this command into the terminal:

curl -sL http://{{ server_host }}/static/client/install.sh | bash
Note: The command above is customized for your server. Make sure your Raspberry Pi can reach {{ server_host }}
3
Answer the Prompts

The installer will ask for:

  • Server URL: Pre-filled with http://{{ server_host }}
  • Device Name: Press Enter to use hostname, or type a custom name
  • Device Serial: Press Enter to auto-detect
4
Wait for Installation

Takes 2-5 minutes. The installer will show progress for each step.

5
Assign Content

Go to Devices page and assign videos or playlists to your new device.

Alternative: Download Script

If you prefer to download and inspect the script first:

Download install.sh

Then run on your Raspberry Pi:

chmod +x install.sh && ./install.sh

What's Next?

  • Device will appear in Devices page
  • Assign videos or playlists to start playback
  • Configure display rotation via Configure button
  • Monitor status on Real-time Dashboard
Need Help?
  • View detailed guide: Quick Start Guide
  • Check device logs: tail -f /home/pi/FFGSA_CSM/logs/player.log
  • Service status: sudo systemctl status FFGSA_CSM-player
